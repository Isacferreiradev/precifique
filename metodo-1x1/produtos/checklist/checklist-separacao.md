# Checklist Separação 1×1

## 1. O Diagnóstico Rápido (O que você faz hoje)
- [ ] Eu tenho apenas uma conta bancária para tudo?
- [ ] Eu pago contas de casa com o cartão da empresa?
- [ ] Eu pago fornecedor com meu PIX pessoal?
- [ ] Eu não sei quanto lucrei mês passado?

*Se marcou qualquer um acima, siga para o passo 2 imediatamente.*

## 2. A Separação (O que você muda HOJE)
- [ ] **Criar/Definir Conta Comercial**: Pode ser um banco digital gratuito (Nubank, Inter, etc). Use ela SÓ para receber de clientes.
- [ ] **Definir Conta Pessoal**: Use sua conta atual ou crie outra SÓ para seus gastos de vida/casa.
- [ ] **A Regra de Ouro**: Todo dinheiro de cliente entra na CONTA COMERCIAL. Nunca na pessoal.

## 3. O Fluxo do Dinheiro (Onde guardar)
- [ ] Cliente pagou -> Vai para Conta Comercial.
- [ ] Precisa pagar fornecedor/custo -> Sai da Conta Comercial.
- [ ] Precisa pagar conta de luz de casa -> Transfere "Salário" da Comercial para Pessoal -> Paga a conta.

## 4. O Juramento (Nunca mais misturar)
- [ ] Prometo nunca pagar uma conta pessoal direto da conta da empresa.
- [ ] Prometo fazer uma transferência única (semanal ou mensal) para minha conta pessoal ("Meu Salário") e viver com esse valor.
